# The Playbook
{: .fs-8 }

Ten phases. Sequential. Do not skip phases or begin implementation before plan stabilization.
{: .fs-5 .fw-300 }

Each phase has a **stop condition** -- an explicit signal that tells you when to advance.

| Phase | Name | Stop Condition |
|:------|:-----|:---------------|
| 0 | [Prerequisites](/flywheel-playbook/playbook/phase-0) | All checklist items complete |
| 1 | [Draft the Plan](/flywheel-playbook/playbook/phase-1) | Comprehensive first draft finished |
| 2 | [Refine the Plan](/flywheel-playbook/playbook/phase-2) | Only minor wording tweaks; no structural improvements |
| 3 | [Alien Artifacts](/flywheel-playbook/playbook/phase-3) | Artifact pass complete; plan version incremented |
| 4 | [Convert to Beads](/flywheel-playbook/playbook/phase-4) | All plan sections represented as beads with dependencies |
| 5 | [QA the Beads](/flywheel-playbook/playbook/phase-5) | Bead changes are only reordering/renaming |
| 6 | [Implement with Swarm](/flywheel-playbook/playbook/phase-6) | All beads implemented and marked complete |
| 7 | [Fresh-Eyes Review](/flywheel-playbook/playbook/phase-7) | Review passes produce no substantive diffs |
| 8 | [Performance](/flywheel-playbook/playbook/phase-8) | All profiling beads complete |
| 9 | [Metacognition](/flywheel-playbook/playbook/phase-9) | Analysis document complete; patterns identified |
