# The Playbook

Ten phases. Sequential. Do not skip phases or begin implementation before plan stabilization.

Each phase has a **stop condition** -- an explicit signal that tells you when to advance.

| Phase | Name | Stop Condition |
|:------|:-----|:---------------|
| 0 | [Prerequisites](phase-0-prerequisites.md) | All checklist items complete |
| 1 | [Draft the Plan](phase-1-draft.md) | Comprehensive first draft finished |
| 2 | [Refine the Plan](phase-2-refine.md) | Only minor wording tweaks; no structural improvements |
| 3 | [Alien Artifacts](phase-3-alien-artifacts.md) | Artifact pass complete; plan version incremented |
| 4 | [Convert to Beads](phase-4-beads.md) | All plan sections represented as beads with dependencies |
| 5 | [QA the Beads](phase-5-qa.md) | Bead changes are only reordering/renaming |
| 6 | [Implement with Swarm](phase-6-swarm.md) | All beads implemented and marked complete |
| 7 | [Fresh-Eyes Review](phase-7-review.md) | Review passes produce no substantive diffs |
| 8 | [Performance](phase-8-performance.md) | All profiling beads complete |
| 9 | [Metacognition](phase-9-metacognition.md) | Analysis document complete; patterns identified |
