# ACFS: From Zero to Hero

**The Agentic Coding Flywheel -- A Complete Guide**

---

## How to Read This Guide

This guide has three levels. Start wherever matches your experience:

| Level | Audience | What You'll Learn |
|-------|----------|-------------------|
| **Strategic** (1.x) | "Never heard of this" | Why the methodology exists, what problem it solves, core principles |
| **Operational** (2.x) | "I want to try it" | Step-by-step: environment, planning, beads, execution, multi-agent |
| **Tactical** (3.x) | "I'm running agents now" | Tool reference, 47 prompts, configuration, scaling, troubleshooting |
